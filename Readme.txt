Unique SEO analysis plugin for your wordpress website.

== Description ==

Chuno SEO Analysis plugin is a unique Wordpress plugin for Wordpress website owners. Once you install Chuno SEO Analysis Plugin you will get following information about your Wordpress website.

Official Website: http://www.chunoa.com

1. Duplicate Titles

2. Duplicate Meta Descriptions

3. Short Meta Descriptions

4. Long Meta Descriptions

5. Robots.txt Information

6. XML Sitemap Information

7. HTTP Status Information

8. Date Information

9. Server Information

10. Connection Information

11. Content Type Information

Note:- I will add more features in upcoming versions.



== Installation ==

1. Upload Chuno SEO Analysis Plugin to WP-Content -> Plugins or Directly Install It from Plugins -> Add New -> Chuno SEO Analysis
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click Chuno SEO Analysis Menu Item in Wordpress Dashboard and Check Report.
